Создание Корпоративного сайта

Корпоративный сайт

Для крупных организаций и представительств, которые занимают лидирующую позицию в своем сегменте рынка, мы предусмотрели разработку корпоративного сайта.

Корпоративный сайт нацелен на привлечение и удержание партнеров по бизнесу, ориентирован на прессу.

Создание корпоративного сайта преследует цель установления деловых связей.

Корпоративный сайт всегда имеет сложную структуру страниц, правильно оптимизированный программный код, включает интерактивные элементы и различные модули.

Для каждого зарегистрированного пользователя на нем предусмотрен личный кабинет.

Качественное создание сайта компании можно выполнить, если воспользоваться нашими услугами. 

Описание

  • Время на разработку

    Индивидуальный дизайн: до 3-х недель.

  • Регистрация в поисковых системах Google, Yandex

    Для того, чтобы Ваш сайт можно было найти в поисковых системах, его необходимо зарегистрировать.

  • Установка системы управления сайтом

    Любому сайту просто необходима система управления для наполнения его какой-либо информацией, изменение функционала, внешнего вида.

  • Дизайн

    Мы разрабатываем для Вас уникальный продающий дизайн. Такой дизайн является кросс-браузерным и адаптирован под любое устройство.

  • Начальная оптимизация сайта

    Мы создаем уже оптимизированные сайты для дальнейшего продвижения в поисковых системах.

  • Наполнение сайта

    Как правило, такой сайт не превышает 20-ти страниц.

  • Установка счетчика

    При помощи счетчика Вы можете просматривать посещаемость ресурса.

  • Онлайн-трансляция

    Мы предоставляем Вам возможность прослеживать за ходом разработки сайта и, при необходимости, Вы сможете сообщить нам о каких-либо корректировках.

  • Бесплатная техническая поддержка на 1 месяц

    Наша компания ценит своих клиентов, поэтому мы дарим Вам техническую поддержку сайта на 1 месяц бесплатно.

  • Сопроводительная документация по системе управления сайтом

    Для Вашего удобства мы написали руководство пользователя по системе управления.

Характеристики

Корпоративный сайт является крупным интернет ресурсом компании и содержит в себе большое количество модулей.

  • Поиск по сайту

    Чтобы найти нужную информацию пользователи зачастую прибегают к поиску по сайту.

  • Новостной блог

    Блог, в котором написаны все последние новости компании, как правило - это небольшой модуль, расположенный вне основного контента.

  • Регистрация пользователей

    Предоставляет доступ к определенным функциям Вашего ресурса.

  • Авторизация пользователей

    Используется для входа на сайт Ваших сотрудников или клиентов.

  • Смена пароля пользователей

    Для улучшения безопасности предусмотрен модуль смены пароля в личном кабинете пользователя.

  • Личный кабинет

    Это личная страничка пользователя, которая предоставляет доступ к каким-либо данным.

  • Галерея

    Галерея - изображения Ваших товаров или услуг.

  • Каталог

    Каталог Вашей продукции. Отображает весь список Ваших товаров в красивой форме для привлечения клиентов.

  • Карта сайта

    Компонент сайта, на котором отображены все страницы в виде древовидной структуры.

  • Отзывы

    Отзывы о ваших товарах или услугах.

  • Строка навигации

    Это путь от начальной страницы до текущей, состоящий из отрезков пути, каждый из которых является разделом сайта.

  • Форма обратной связи

    Такая форма нужна для получения информации от клиентов (один из способов связи).

  • Дополнительные модули

    Установка дополнительных модулей по Вашему желанию (в зависимости от дополнительных компонентов может измениться цена).
